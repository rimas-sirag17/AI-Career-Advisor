# AI Career Advisor
An intelligent system that analyzes user skills and educational background to recommend suitable career paths, job roles, and personalized learning resources. It uses AI-based skill matching and suggests courses to fill skill gaps.

# Features
- Recommend top job matches based on user skills
- Identify missing skills for each job
- Suggest relevant courses to improve skills
- Interactive web interface using Streamlit
# Usage Example
Enter your skills separated by commas in the input box (e.g., Python, Excel, SQL)
Click "Analyze My Career"
View top job matches, missing skills, and recommended courses
